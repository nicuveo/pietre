Piètre is a small stack-based language, developed with three goals:

- provide a good resource to learn about compilers and tooling
- provide an example of a medium-size Haskell project
- facilitate working with stack-based esoteric languages

Specifically, the aim is for the language to be directly translatable to [Piet](https://www.dangermouse.net/esoteric/piet.html), an esoteric language in which the source code is an image.
